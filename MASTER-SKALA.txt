# SKALA HOME SERVICES PRO PREMIUM

## MASTER DEL PROYECTO

### CONCEPTO

SKALA Home Services es una plataforma SaaS para gestión de mantenimiento inmobiliario.

Conecta:

* Inmobiliarias
* Administradores de edificios
* Técnicos
* Equipo interno SKALA

Todo centralizado desde una única plataforma.

---

# ESTADO ACTUAL

## Arquitectura

Proyecto modular.

Separado por:

* HTML
* CSS
* JavaScript
* Assets

Preparado para:

* LocalStorage (fase actual)
* PostgreSQL (fase siguiente)
* Backend Node.js
* API REST

# REQUERIMIENTOS

Cada requerimiento debe contener:

* ID único
* Inmobiliaria
* Propiedad
* Dirección
* Categoría
* Descripción
* Prioridad
* Fecha creación
* Estado
* Técnico asignado
* Evidencias fotográficas

Estados:

* Pendiente
* En revisión
* Asignado
* En ejecución
* Finalizado
* Cerrado

---

# SISTEMA DE FOTOGRAFÍAS

Futuro:

# REDISEÑO DEL CHAT (PRIORIDAD ALTA)

## PROBLEMA

Actualmente existe un único chat.

Esto no sirve para producción.

---

## SOLUCIÓN

Cada conversación debe estar vinculada a un requerimiento.

Ejemplo:

REQ-1001
Fuga de agua
Edificio Santa María

Chat independiente.

---

REQ-1002
Problema eléctrico
Torre Norte

Chat independiente.
